# digitallogisim
Um Half-Subtractor (ou Meio Subtrator) é um circuito combinacional digital usado para subtrair dois bits binários. Ele produz duas saídas:

Diferença (D) – resultado da subtração do bit subtraendo do bit minuendo.

Empréstimo (B ou Borrow) – indica se foi necessário “emprestar” 1 da próxima posição de bit, caso o minuendo seja menor que o subtraendo.
